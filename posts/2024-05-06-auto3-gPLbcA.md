企业级 SecOps 服务由 CyberProof 利用 Google Chronicle Security Operations 迅速检测和应对网络事件。![图片](https://ai-techpark.com/wp-content/uploads/2024/05/CyberProof-960x540.jpg){ width=60% }

---

CyberProof，一家 UST 公司，宣布与 Google Cloud 扩展伙伴关系，重点利用 Google Chronicle Security Operations 和其他 Google Cloud 安全解决方案来扩展 CyberProof 的托管 XDR 服务的能力。凭借这一合作伙伴关系，CyberProof 扩展了其安全服务提供，帮助客户跨越组织和功能隔阂，提供端到端的服务体验。
CyberProof 的集成安全平台可可视化和缓解整个企业的网络风险。随着越来越多的企业将资产转移到云端，CyberProof 优化安全姿势和事件监控，以尽量减少整体网络风险并降低成本。Google Chronicle Security Operations 是 Google Cloud 安全全面组合中的一部分，结合了 Google Cloud 的先进功能和深入的威胁情报，实现了由 GenAI 完全升级的情报驱动的安全运营。CyberProof 团队提供了一个端到端解决方案，将 Google Chronicle Security Operations 纳入其安全运营中心 (SOC) 服务和其他服务中。
CyberProof 防御中心 (CDC) 与 Google Chronicle Security Operations 的安全事件和事件管理 (SIEM) 以及安全编排、自动化和响应 (SOAR) 能力一起，帮助扩展 CyberProof 提供客户期望从 AI 动力和自适应托管扩展检测和响应 (MXDR) 服务中获得的价值和体验的能力。
“安全服务提供商提供深度工程能力帮助客户实施面向未来的安全架构尤为重要。减少网络风险和成本的机会是无限的，但存在许多未知因素，错误代价高昂。我们与 Google Cloud 的战略关系有助于更好地满足客户需求并迅速推动创新发展，”CyberProof 首席执行官 Tony Velleca 表示。
“CyberProof 和 Google Cloud 正在合作推出面向云规模速度和有效性建立的情报驱动型企业级网络安全解决方案。CyberProof 在处理大型和复杂生态系统以及深入了解应对威胁的知识方面的专业知识对帮助我们的客户管理企业安全至关重要，”Google Cloud 的合作伙伴副总裁 Magali Bohn 表示。
CyberProof 帮助大型跨国组织迁移到云端的专业知识与 Google Cloud 对未来安全的愿景一致。支持 GenAI 等新兴功能的面向未来的安全架构对于增强网络安全服务至关重要。CyberProof 利用共管安全运营服务模式，同时整合 Google 的 Chronicle SOAR 进行服务交付。这种服务模式支持以安全结果为重点的数字转型。
有关 CyberProof 为 Google 提供的托管 XDR 的更多信息，请访问 CyberProof 网站此处：https://www.cyberproof.com/security-services/managed-xdr-for-google/。
